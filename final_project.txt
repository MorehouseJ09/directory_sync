# SYNC PROGRAM

1.) Get file list for both initial directory and final directory
2.) Flag files to be deleted from destination
3.) Compare continuing files that need to be synced
4.) copy files to the proper locations -- overwrite

Modules

-	file list -- returns a recursive list of all files in the directory


Directory - class -- this is the master object that holds all of the files

	-	delete a file -- both relative / absolute
	-	add a file  -- both relative / absolute
	-	get files -- gets all the files
